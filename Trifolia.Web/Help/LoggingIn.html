Logging In

Getting Started ››

To use the Trifolia Workbench, you need access credentials.

Once you log in to Trifolia Workbench, the functions you perform depend on the access permissions granted to your login credentials.

If it is the first time logging into Trifolia, you will be prompted to enter "My Profile" information. See My Profile for more information.

If you have logged into Trifolia before, you can modify your My Profile information from the menu in the top-right corner of Trifolia.

Log In as a Commercial User

  1. In the upper-right corner, select Log In and choose Login from the drop-down menu. The Log In page appears, as show above.
  2. Enter your credentials and select the Login button. To log in automatically each time you visit Trifolia, select the Keep me logged in checkbox.
  3. Confirm that you are a human (and not a robot) by completing the re-captcha verification
    1. Re-captcha verification is performed by Google's re-captcha API. More information on Google's re-captcha can be found here.

Log in as an HL7 User

  1. From either the Trifolia home page or the commercial account login page, select the "here" link at the end of the "HL7 Members can login and non-members can register to use Trifolia Workbench here." sentence
  2. Trifolia will redirect you to the HL7 login page
  3. Login to HL7 with your HL7 credentials
  4. HL7's website will redirect you back to Trifolia, as a logged-in user

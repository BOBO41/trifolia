Use the Templates page to view a list of templates. Use the View Template page to view the template details.

Viewing a Template

There are two ways to view a template:

  • From the Implementation Guides list
  • From the Templates list

To view a template from the Implementation Guides list

  1. From the Browse menu, choose Implementation Guides.
  2. Use the Trifolia's Automatic Lookup feature to find the Implementation Guide containing the template you want to view. A list of Implementation Guides that match your search criteria appears.
  3. In the Implementation Guides list, click View for the Implementation Guide you want to view.The Implementation Guide appears, showing you a list of templates contained in the guide.
  4. Pop down the list of templates for the type of template you want to view. Locate the template you want to view and click View. The template viewer opens, containing the template details.

To view a template from the Templates list

  1. From the Browse menu, choose Templates.
  2. Use the Trifolia's Automatic Lookup feature to find the templates you want to browse. A list of templates that match your search criteria appears.
  3. In the Template list, choose View for the desired template. The template viewer opens, containing the template details.

Viewing the Template

  • Constraints  - view a list of the template's constraints as they appear in the Implementation Guide.
  • Samples - view a code sample attached to the template. The "Samples" tab does not appear in the template viewer if no samples exist for the template.
  • Relationships - view the relationship of the current template to other templates.
    • Implied By the current template - lists any templates that reference the current template
    • Contains a template - lists any templates contained within constraints in the current template
    • Contained By a template - lists any templates that reference the current template
    • Implies a template - lists a single template referenced in the Implies field of the current template
  • Changes - View changes to the template. View the changes Inline or as a List. (See below)

Viewing Changes - List

When you view changes to a template as a List, each constraint change is listed in a table. The type of change is listed as:

  • Added
  • Modified
  • Removed

Notes:

  • Unchanged constraints do not appear in the list.
  • Each entry contains the constraint reference number, the old narrative, and the new narrative.
